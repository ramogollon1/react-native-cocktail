# Cocktail App

### Before run the app

#### 1. Install cocoapods
- `sudo gem install cocoapods`


#### 2. Run pods inside folder ios
- `cd ios && pod install`

### Run the following commands from the project folder

### Content

* ios
* android
* src
  * app
    * assets
    * components
    * screens
  * constants
  * redux
    * search
  * services

### Install dependencies

- `npm install or npm i`

### Run on Androind

- `npm run android`

### Run on IOS

- `npm run ios`

### Run tests

- `npm test`
