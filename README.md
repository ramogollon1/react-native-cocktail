# Cocktail App

### Before run the app

#### 1. Install dependencies

- `npm install or npm i`
#### 2. Install cocoapods
- `sudo gem install cocoapods`

#### 3. Run pods inside folder ios
- `cd ios && pod install`
### Content

```
├── ios
├── android
├── src
  ├── app
    ├── assets
    ├── components
      ├── GradientContainer
      ├── Item
      ├── ListItem
    ├── screens
      ├── Home
      ├── Search
  ├── constants
  ├── redux
    ├── search
  ├── services
```

### Start Metro to listen the application

- `npx react-native start`
### Run on Androind

- `npm run android`

### Run on IOS

- `npm run ios`

### Run tests

- `npm test`
